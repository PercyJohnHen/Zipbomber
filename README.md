# 🚀 ZipBomber

A simple Python command-line tool that processes a number using a single command.

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python)

## 📖 Overview

**ZipBomber** is a lightweight Python CLI application that takes a number as input and creates a zipbomb with a bomb size of that number in **GB**.


## ▶️ Usage

Run the program with:

```bash
python zb.py -s <file size in **GB**>
```

### Example

```bash
python zb.py -s 120
```

### Example(using the binary)

```bash
zb.exe -s 120
```

### Use the -h for help
```bash
zb.exe -h or zb.exe --help
```
